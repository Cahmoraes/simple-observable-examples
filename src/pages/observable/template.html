Sintaxe

const $observable = so.observable(param [,function])

Cria um observable.

Parâmetros

param: String, Number, Object, null, Function.
Valor inicial do observable.

Métodos

prevValue() => retorna o valor anterior observable.
getInitialValue() => retorna o valor inicial do observable.

const $observable = so.observable(1)
$observable(3)
$observable(5)
console.log($observable.prevValue()) // 3 console.log($observable.getInitialValue()) // 1
