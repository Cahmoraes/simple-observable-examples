Cria um Pipeable observable.

Pipeable Observable, é um observable que lembra das funções que transformarão seu valor. Toda vez que seu valor for atualizado, as funções pipeables serão executadas sobre o novo valor.

Parâmetros

param: Função ou lista de funções.
Funções que vão transformar o valor do Observable sempre que ele for atualizado.
Essas funções devem receber um parâmetro (valor atual do observable) e retornar o valor transformado.

Métodos

prevValue() => retorna o valor anterior observable.
getInitialValue() => retorna o valor inicial do observable. 

                const $pipeExample_1 = so.observable('caique').pipe(

                (value) => value.toUpperCase()

                const $pipeExample_2 = so.observable(10).pipe(
                  (value) => Number(value), 
                  (value) => value.toLocaleString('pt-BR', { style: 'currency', currency: 'BRL' }) 
                ) 

                $pipeExample_2.subscribe(console.log) 

                $pipeExample_2() // R$ 10,00 

                $pipeExample_2(250) // R$ 250,00 

                $pipeExample_2('3000') // R$ 3.000,00 

                console.log($pipeExample_2.prevValue()) // R$ 250,00
