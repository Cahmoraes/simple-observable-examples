Cria um array de observables.

Cada obervable alterado ou removido reflete no array.

Se for passado um array como parâmetro no construtor, cada elemento do array será convertido para um elemento no Observable Array.

Pode passar quantos elementos forem necessários, cada elemento será convertido para um elemento no Observable Array.

Parâmetro

Arrays: Se for passado um array, cada elemento do array será convertido para um elemento do Observable Array.

Se nada for passado, será criado um Array vazio

                $array.add('novo elemento')

Métodos

add(param): Adiciona um novo elemento no array.
Se o array estiver vazio e param for um array, cada item de param será adicionado como um item do array.

remove(param): Remove elemento do Array.
param: int, object, function.
Se param for um número inteiro, removerá o índice do array do reepectivo param.
Se param for uma função, deverá ser um elemento do array ex.: $array()[0].
Se param for um objeto, deverá ser passado como ex.: $array()[0]() 

                const $array = so.observableArray([1, 2, 3, 4]) 

                $array.remove(1) // remove number 2 
 
                $array.remove($array.get(3)) // remove number 4

get(index): Retorna elemento do array, do índice especificado.
Nota: Retorna como observable.

getValue(index): Retorna elemento do array, do índice especificado.
Retorna o valor do elemento do índice especificado. 

                const $array = so.observableArray([1, 2, 3, 4]) 

                console.log($array.get(2)) // function observable(newParamValue) {...} 

                console.log($array.getValue(2)) // 3
